# Chess Review System

A MERN stack web application for reviewing and analyzing chess games. Upload your games, review them move-by-move, add annotations, and get engine-powered analysis.

## Architecture

This project follows a MERN (MongoDB, Express, React, Node.js) architecture

## Development Roadmap

- **Milestone 0** (Current): Project skeleton and tooling
- **Milestone 1**: Basic game model and PGN storage
- **Milestone 2**: Move list and navigation
- **Milestone 3**: Chessboard visualization
- **Milestone 4**: Manual annotations
- **Milestone 5**: Authentication
- **Milestone 6**: Engine analysis (Stockfish)
- **Milestone 7**: Game list and filters
- **Milestone 8**: Sharing and export
- **Milestone 9**: Chess.com/Lichess integration
- **Milestone 10**: Integration improvements

## Tech Stack

- **Backend**: Node.js, Express.js, MongoDB (Mongoose)
- **Frontend**: React, JavaScript, Vite, React Router
- **Engine**: Stockfish

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- MongoDB (local installation or MongoDB Atlas account)
- npm or yarn
